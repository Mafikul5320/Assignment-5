Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML shows or changes the HTML inside an element, including tags. innerText gives only the visible text, while textContent gives all text, even if it's hidden.

Question-3:What is event delegation in the DOM?

Answer: Event delegation means adding an event listener to a parent element instead of many child elements. When a child element is clicked, the event "bubbles up" to the parent, which handles it. This makes the code faster and works even if new child elements are added later.

Question-4:What is event bubbling in the DOM?

Answer: Event bubbling means that when you click on a child element, the event first happens there and then moves up to its parent, grandparent, and so on. For example, if you click a button inside a div, first the button reacts, then the div, then the whole page. This helps parents handle events from their children.

Question-5:How do you create, add, and remove elements using JavaScript?

Answer:I create an element using document.createElement("tag"). To add it, use parent.appendChild(newElement) or parent.append(newElement). To remove it, use parent.removeChild(element) or element.remove().
